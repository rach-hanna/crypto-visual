# crypto visual  
this is a visual dashboard of short term BTC/USDT liquidity/market dynamics and volatility using candlesticks, volume, realised volatility and order-book depth

## demo

[View Live Dashboard](https://rach-hanna.github.io/crypto-visual/dashboard.html)

## purpose  
the tool allows me to visualise a snapshot of crypto liquidity and market microstructure. it calculates simple liquidity metrics like spread and volatility.
