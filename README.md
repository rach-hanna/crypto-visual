# cryptoflows  
this is a visual dashboard of short term BTC/USDT liquidity and volatility via binance API and plotly. coded using python, it uses candlesticks, volume, realised volatility and order-book depth to showcase market dynamics.


## purpose  
the tool allows me to visualise crypto liquidity and market microstructure in real-time. it calculates simple liquidity metrics like spread and volatility. interactive display, used for market analysis.

2. run the app
